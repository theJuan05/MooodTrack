Final Requirement in ITEP 308 – System Integration and Architecture I First Semester, Academic Year 2025-2026 
“Web Application with Machine Learning Integration (PHP + Composer + Packagist)” Project Overview 
Your final project involves developing a PHP-based Web Application that demonstrates System  Integration by integrating at least one Machine Learning (ML) library from Packagist.org using  Composer. 
You must follow the Design Thinking methodology (Hills, Sponsor User, Playback) and  demonstrate the working system through a 10-minute recorded video. 
A PowerPoint presentation (Canva or PowerPoint) is also required.
