# Mood Tracker Web Application with Machine Learning

## Project Overview
This project is a PHP-based web application that allows users to track their daily moods.
It integrates a Machine Learning library using Composer to automatically classify mood
sentiments.

## Machine Learning Library
- Library: php-ai/php-ml
- Algorithm: Naive Bayes Classifier
- Installed via Composer
